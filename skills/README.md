# Skills

This repository contains local skills used by coding agents.

## Available Skills

- `skills/changelog/SKILL.md`: maintain root `CHANGELOG.md` using git history (`commit date`, `commit message`, and short hash).
- `skills/docker-deployment/SKILL.md`: deploy application stack with Docker Compose, including Docker readiness checks and failure diagnostics.
- `skills/docker-deployment/SKILL.md`: Build and deploy Kanon Platform with Docker.
